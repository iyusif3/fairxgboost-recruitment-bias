FairXGBoost: Fairness-Aware Recruitment Bias Mitigation

Engineered fairness-aware loss function for XGBoost, developed as part of a 
research proposal at the Department of Computer Science, KNUST.

Researcher: Imran Yusif (20905589)  
Supervisor: Dr. Eric Opoku Osei

Status
Repository created in advance of data collection and experimentation, per 
reproducibility requirements. Code and notebooks will be added as the 
project progresses through the phases below.

 Planned structure
- `notebooks/` — preprocessing, baseline training, FairXGBoost engineering, evaluation
- `data/` — anonymised data (added post-ethics clearance, if shareable)
- `results/` — model checkpoints, metrics, visualisations
- `src/` — fairness metric and preprocessing utility modules
